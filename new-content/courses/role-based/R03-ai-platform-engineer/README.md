# R03: AI Platform Engineer Specialization

## Overview

Focused on the backbone of the AI stack, ensuring systems run reliably at scale. This course covers container orchestration, GPU infrastructure management, inference optimization, and the operational expertise needed to serve AI models in production with high availability and cost efficiency.

## Prerequisites

- Beginner course track (B01-B10) completed
- Intermediate course track (I01-I03) completed
- Strong Linux and systems administration skills
- Familiarity with Docker, Kubernetes, and cloud infrastructure
- Basic understanding of ML model inference

## Target Role

AI Platform Engineers build and maintain the infrastructure that powers AI systems at scale. They manage GPU clusters, optimize inference performance, implement autoscaling and reliability patterns, and ensure cost-efficient operation of model serving infrastructure. This role requires deep systems knowledge combined with an understanding of AI-specific workload characteristics.

## Modules

| Module | Name | Lessons | Key Topics |
|--------|------|---------|------------|
| M01 | Serving & Orchestration | 3 | Container orchestration, autoscaling, SLOs, secret management, access control |
| M02 | GPU & Inference Infrastructure | 3 | GPU memory management, KV cache optimization, VLLM and Triton deployment |
| M03 | Inference Optimization at Scale | 3 | Quantization, speculative decoding, cost optimization and performance tuning |

## Key Tools & Technologies

- Docker and Kubernetes
- NVIDIA GPU drivers and CUDA
- VLLM for LLM serving
- NVIDIA Triton Inference Server
- Prometheus / Grafana for monitoring
- HashiCorp Vault for secret management
- Load testing tools (Locust, k6)

## Learning Outcomes

- Deploy and manage containerized AI workloads with GPU-aware scheduling on Kubernetes
- Design autoscaling strategies and reliability patterns that meet defined SLOs for AI systems
- Optimize GPU memory usage, implement KV cache strategies, and deploy models with VLLM and Triton
- Apply quantization and speculative decoding techniques to reduce inference cost and latency
- Perform capacity planning and cost optimization for large-scale inference infrastructure
