# L01: Production-Grade RAG System

## Overview

Build a complete production-grade RAG system that combines hybrid retrieval, citation enforcement, and comprehensive evaluation. This is the first part of the capstone project, focusing on the core retrieval and generation pipeline.

## Topics Covered

- End-to-end RAG system architecture for production
- Hybrid retrieval with BM25 + vector search integration
- Citation enforcement in the generation pipeline
- Golden dataset creation for evaluation
- Evaluation harness design and implementation

## Lab: Building a Complete Production-Grade RAG System with Evaluation (Capstone Part 1)

Build a fully functional production-grade RAG system from scratch, integrating hybrid search, citation enforcement, and a comprehensive evaluation harness with a curated golden dataset.

### Lab Objectives

- Design and implement an end-to-end RAG architecture with hybrid retrieval
- Integrate citation enforcement that grounds every generated claim in source documents
- Curate a golden dataset of queries, expected retrievals, and reference answers
- Build an evaluation harness that measures retrieval quality, answer accuracy, and citation correctness

## Key Takeaways

- Production RAG systems require careful architecture design that balances quality, latency, and cost
- Citation enforcement transforms RAG from a convenience tool into a trustworthy information system
- A well-curated golden dataset is the single most valuable asset for maintaining system quality
- End-to-end evaluation (retrieval + generation + citation) reveals issues that component-level testing misses
