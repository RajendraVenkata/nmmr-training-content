# M04: Capstone Project

## Overview

Apply all skills learned throughout the AI Engineer specialization to build a complete production-grade RAG system with full monitoring and observability. This two-part capstone integrates hybrid retrieval, citation enforcement, evaluation harnesses, and CI/CD quality gates into a single cohesive project.

## Prerequisites

- R01-M01: Advanced RAG Engineering (completed)
- R01-M02: Agent Development (completed)
- R01-M03: Production Readiness (completed)

## Lessons

| Lesson | Name | Has Lab? | Key Topics |
|--------|------|----------|------------|
| L01 | Production-Grade RAG System | Yes | End-to-end RAG with hybrid retrieval, citation enforcement, evaluation, golden dataset |
| L02 | Monitoring & Observability Layer | Yes | Dashboards, latency tracking, CI/CD integration, regression gates |

## Key Tools

- LangChain / LangGraph
- Vector databases + BM25 hybrid search
- Prometheus / Grafana
- GitHub Actions or GitLab CI
- Python evaluation frameworks

## Learning Outcomes

- Build an end-to-end production-grade RAG system combining hybrid retrieval and citation enforcement
- Create a comprehensive golden dataset and evaluation harness for continuous quality tracking
- Add full observability with dashboards, latency tracking, and error monitoring
- Integrate evaluation harnesses into CI/CD pipelines with automated regression gates
