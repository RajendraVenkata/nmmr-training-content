# R01: AI Engineer Specialization

## Overview

The most in-demand AI role in 2026. This course focuses on building end-to-end features powered by LLMs, covering advanced RAG engineering, agent development, production readiness, and culminating in a comprehensive capstone project.

## Prerequisites

- Beginner course track (B01-B10) completed
- Intermediate course track (I01-I03) completed
- Proficiency in Python and familiarity with LLM APIs
- Understanding of RAG fundamentals and agent concepts

## Target Role

AI Engineers build LLM-powered features and systems end-to-end. They design retrieval pipelines, develop agentic workflows, implement evaluation harnesses, and ensure production reliability. This role sits at the intersection of software engineering and applied AI, requiring both strong coding skills and deep understanding of LLM behavior.

## Modules

| Module | Name | Lessons | Key Topics |
|--------|------|---------|------------|
| M01 | Advanced RAG Engineering | 3 | Domain-specific RAG, hybrid search, citation enforcement |
| M02 | Agent Development | 3 | LangGraph workflows, tool routing, memory and state management |
| M03 | Production Readiness | 3 | Latency/cost/reliability, evaluation harnesses, monitoring and regression gating |
| M04 | Capstone Project | 2 | Production-grade RAG system with monitoring and observability |

## Key Tools & Technologies

- LangChain / LangGraph
- Vector databases (ChromaDB, Pinecone, Weaviate)
- BM25 and hybrid search engines
- Python evaluation frameworks
- Prometheus / Grafana for monitoring
- CI/CD pipelines with eval gates
- OpenAI, Anthropic, and open-source LLM APIs

## Learning Outcomes

- Design and build domain-specific RAG systems with hybrid retrieval and citation enforcement
- Develop production agentic workflows using LangGraph with tool routing and persistent memory
- Implement evaluation harnesses with golden datasets for continuous quality tracking
- Set up production monitoring with regression gating and automated quality gates
- Build a complete production-grade RAG system with full observability and CI/CD integration
