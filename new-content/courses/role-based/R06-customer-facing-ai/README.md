# R06: Customer-Facing AI Roles Specialization

## Overview

Covers high-leverage positions that combine coding skills with user interaction. This course prepares engineers for Forward Deployed Engineer, Technical Account Manager, and Developer Advocate roles -- positions where technical depth meets customer empathy.

## Prerequisites

- Beginner course track (B01-B10) completed
- Intermediate course track (I01-I03) completed
- Strong communication and presentation skills
- Proficiency in Python and/or JavaScript
- Understanding of LLM APIs, RAG, and agentic AI concepts

## Target Role

Customer-Facing AI professionals operate at the intersection of engineering and customer success. Forward Deployed Engineers deploy AI directly into customer workflows. Technical Account Managers guide customers through post-adoption optimization. Developer Advocates build demos, create content, and grow developer communities. All three tracks require strong technical foundations combined with the ability to communicate complex AI concepts clearly.

## Modules

| Module | Name | Lessons | Key Topics |
|--------|------|---------|------------|
| M01 | Forward Deployed Engineer | 3 | Customer workflow deployment, data troubleshooting, system design & API integration |
| M02 | Technical Account Manager for AI | 3 | Post-adoption advisory, production optimization, customer success metrics |
| M03 | AI Developer Advocate | 5 | Technical foundations, demo apps, content creation, DevRel strategy, personal brand |

## Key Tools & Technologies

- Python / JavaScript / TypeScript
- Gradio / Streamlit for demo applications
- LLM APIs (OpenAI, Anthropic, open-source)
- REST API design and integration tools
- Content creation tools (blog platforms, video recording)
- Analytics platforms for customer success metrics
- Presentation and communication tools

## Learning Outcomes

- Deploy AI solutions into real customer workflows, handling messy data and edge cases
- Design reliable API integrations for customer-facing AI systems
- Guide customers through AI optimization for latency, cost, and reliability improvements
- Define and track customer success metrics and AI adoption KPIs
- Build polished demo applications using Gradio or Streamlit
- Create technical content (tutorials, documentation, videos) that drives developer adoption
- Develop a DevRel strategy covering awareness, activation, and retention
- Build a personal brand as a subject matter expert in AI
