# M02: Technical Account Manager for AI

## Overview

Prepares engineers for the Technical Account Manager role, where the focus shifts from initial deployment to long-term customer success. This module covers post-adoption advisory, production optimization guidance, and defining the metrics that prove AI value to customers.

## Prerequisites

- M01: Forward Deployed Engineer completed (or equivalent field experience)
- Understanding of production AI system architecture
- Familiarity with performance optimization concepts (latency, cost, reliability)
- Strong communication and relationship-building skills

## Lessons

| Lesson | Name | Has Lab? | Key Topics |
|--------|------|----------|------------|
| L01 | Post-Adoption Advisory | No | Scaling AI post-deployment, optimization opportunities, long-term customer relationships |
| L02 | Production AI Optimization (Latency & Cost) | No | Latency reduction, cost optimization, performance tuning, monitoring setup |
| L03 | Customer Success Metrics | No | AI adoption metrics, ROI measurement, usage analytics, customer health scoring |

## Key Tools

- Performance monitoring tools (Datadog, New Relic, Grafana)
- Cost analysis dashboards
- Customer health scoring frameworks
- Analytics platforms
- Presentation and reporting tools

## Learning Outcomes

- Guide customers through scaling AI systems beyond initial deployment
- Identify and recommend optimization opportunities for latency and cost reduction
- Set up performance monitoring and alerting for customer AI deployments
- Define and track meaningful customer success metrics for AI products
- Build and maintain long-term advisory relationships with technical stakeholders
