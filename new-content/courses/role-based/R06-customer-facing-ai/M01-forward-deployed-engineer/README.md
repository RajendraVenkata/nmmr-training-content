# M01: Forward Deployed Engineer

## Overview

Prepares engineers to work directly with customers, deploying AI into real-world workflows where data is messy and requirements are ambiguous. This module covers the practical skills needed to bridge the gap between AI capabilities and customer environments, from troubleshooting data issues to designing robust integrations.

## Prerequisites

- R06 course prerequisites (B01-B10, I01-I03 completed)
- Strong Python skills and API development experience
- Understanding of LLM capabilities and limitations
- Comfort working in ambiguous, customer-facing environments

## Lessons

| Lesson | Name | Has Lab? | Key Topics |
|--------|------|----------|------------|
| L01 | Deploying AI in Customer Workflows | No | Customer workflow analysis, real-world deployment, adapting to customer environments |
| L02 | Troubleshooting Messy Data | No | Edge case debugging, data quality assessment, quick fixes vs long-term solutions |
| L03 | System Design & API Integration | Yes | AI integration design, API design for customers, system architecture for reliability |

## Key Tools

- Python / JavaScript
- REST API frameworks (FastAPI, Express)
- Data profiling and quality tools
- Logging and monitoring systems
- Customer communication tools

## Learning Outcomes

- Analyze customer workflows to identify where AI adds the most value
- Deploy AI solutions into messy, real-world customer environments
- Diagnose and resolve data quality issues in customer data pipelines
- Design reliable API integrations that meet customer system requirements
- Communicate technical issues and trade-offs effectively to non-technical stakeholders
