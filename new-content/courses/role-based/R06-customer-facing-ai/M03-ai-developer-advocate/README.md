# M03: AI Developer Advocate

## Overview

Covers the complete skill set needed for AI Developer Advocacy, from deep technical foundations through demo building, content creation, community strategy, and personal branding. This module prepares engineers to become the public face of AI platforms, driving developer adoption through education and engagement.

## Prerequisites

- M01: Forward Deployed Engineer completed (or equivalent customer-facing experience)
- Strong coding skills in Python and/or JavaScript
- Understanding of LLMs, RAG, fine-tuning, and agentic AI systems
- Comfort with public speaking and written communication

## Lessons

| Lesson | Name | Has Lab? | Key Topics |
|--------|------|----------|------------|
| L01 | Technical Foundations for Advocacy | No | Deep technical baseline, explaining production trade-offs clearly |
| L02 | Building Demo Applications (Gradio/Streamlit) | Yes | Compelling demos, Gradio, Streamlit, making AI accessible through interactive interfaces |
| L03 | Content Creation & Tutorials | No | Documentation, tutorials, video demos, sharing insights on new releases |
| L04 | DevRel Strategy & Community Building | No | Awareness-activation-retention, workshops, launch playbooks, community growth |
| L05 | Building Your Personal Brand | No | Subject matter expertise, thought leadership, scaling to senior roles |

## Key Tools

- Gradio / Streamlit for demo applications
- Blog platforms (Dev.to, Medium, Hashnode)
- Video recording and editing tools
- Social media platforms (LinkedIn, Twitter/X, GitHub)
- Community management tools (Discord, Slack)
- Analytics for measuring content and community impact

## Learning Outcomes

- Explain complex AI concepts and production trade-offs clearly to diverse technical audiences
- Build polished, interactive demo applications that showcase AI capabilities effectively
- Create high-quality technical content including tutorials, documentation, and video demos
- Design and execute a DevRel strategy covering awareness, activation, and retention
- Build a personal brand as a recognized subject matter expert in AI
