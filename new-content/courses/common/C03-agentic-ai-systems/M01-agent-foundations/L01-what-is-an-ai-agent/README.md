# L01: What is an AI Agent - Beyond Chatbots

## Overview

This lesson defines what AI agents are, how they fundamentally differ from chatbots, and why autonomy is the core distinguishing characteristic. You will learn the perception-reasoning-planning-action cycle that underpins all agent behavior and explore the spectrum from reactive to planning-based agents.

## Topics Covered

- Defining AI agents vs chatbots
- Autonomy as the core distinction between agents and chatbots
- The perception-reasoning-planning-action cycle
- Reactive agents and their characteristics
- Planning agents and multi-step reasoning
- The spectrum of agent autonomy

## Key Takeaways

- An AI agent is defined by its ability to autonomously perceive its environment, reason about goals, plan actions, and execute them without step-by-step human instruction
- Chatbots are stateless request-response systems, while agents maintain state and pursue goals across multiple steps
- The perception-reasoning-planning-action cycle is the foundational loop that drives all agent behavior
- Reactive agents respond immediately to inputs without planning ahead, making them faster but less capable for complex tasks
- Planning agents decompose problems into sub-goals and execute multi-step strategies, trading speed for capability
