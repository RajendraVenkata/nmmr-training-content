# From Static RAG to Agentic RAG

## Overview

Understand the limitations of static single-retrieval RAG and how agentic RAG extends traditional systems by allowing the agent to autonomously formulate queries, critique results, and perform iterative refinement.

## Topics Covered

- Limitations of static single-retrieval RAG
- How agentic RAG extends traditional RAG with autonomy
- Autonomous query formulation by the agent
- Self-critique of retrieval results
- Iterative refinement for higher quality answers

## Key Takeaways

- Static RAG performs a single retrieval step which may miss relevant information
- Agentic RAG allows the agent to iterate, critique, and refine its own retrieval process
- The shift from static to agentic RAG dramatically improves answer quality for complex queries
