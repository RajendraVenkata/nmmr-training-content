# smolagents - Code-First Agents

## Overview

Explore smolagents from Hugging Face, a lightweight framework where agents write and execute Python code blocks directly rather than using JSON-based tool calls, enabling more expressive and accurate complex operations.

## Topics Covered

- Code agents vs JSON-based tool call agents
- Writing and executing Python code blocks as actions
- Expressiveness advantages for complex operations
- Hugging Face ecosystem integration

## Lab: Building a Code-First Agent

Build an agent using smolagents that performs complex operations by generating and executing Python code.

### Lab Objectives

- Set up smolagents and configure a code-first agent
- Implement tool execution via Python code generation
- Compare output quality and accuracy vs JSON-based approaches
- Handle code execution errors and safety boundaries

## Key Takeaways

- Code agents are often more expressive and accurate than JSON-based tool calling for complex tasks
- smolagents provides a lightweight alternative to heavier frameworks
- Code-first agents can compose operations more naturally than structured JSON calls
