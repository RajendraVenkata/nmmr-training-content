# L02: The Five Portfolio Projects That Matter

## Overview

This lesson identifies the five specific project types that carry the most weight in AI hiring decisions. Each project type demonstrates a distinct competency that employers actively seek, from retrieval systems to real-time multimodal applications, giving you a concrete roadmap for building a differentiated portfolio.

## Topics Covered

- Production-grade RAG system (end-to-end retrieval pipeline with evaluation)
- Local AI Assistant using Small Language Models (SLM)
- Monitoring and Observability Layer for AI systems
- Specialized Fine-Tuning Project (domain adaptation with measurable improvement)
- Real-Time Multimodal Application (combining text, image, audio, or video)
- How each project maps to a specific employer need
- Prioritizing which projects to build first based on your target roles

## Key Takeaways

- A production-grade RAG system proves you can build retrieval pipelines beyond basic vector search
- A Local AI Assistant demonstrates you understand inference optimization, hardware constraints, and privacy-first design
- A Monitoring and Observability Layer shows you think about what happens after deployment, not just before
- A Specialized Fine-Tuning Project proves you can improve model performance on domain-specific tasks with measurable metrics
- A Real-Time Multimodal Application signals that you can work with cutting-edge capabilities and handle complex input/output pipelines
