# The Five Portfolio Projects That Matter

## Overview

Learn about the five specific portfolio projects that demonstrate production-grade AI expertise to hiring managers: RAG systems, local AI assistants, monitoring layers, fine-tuning projects, and multimodal applications.

## Topics Covered

- Production-Grade RAG System (hybrid retrieval, citations, evaluation with Ragas)
- Local AI Assistant with Small Language Models (Ollama, benchmarking, JSON validation)
- Monitoring & Observability Layer (tracing, dashboards, CI/CD regression gating)
- Specialized Fine-Tuning Project (SFT, DPO, before/after metrics)
- Real-Time Multimodal Application (streaming, latency budgeting, resilience)

## Key Takeaways

- Each project targets a different dimension of production AI expertise
- Projects should include evaluation, monitoring, and measurable quality metrics
- Together these five projects cover the full AI engineering lifecycle
