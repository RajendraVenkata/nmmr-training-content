# Building in Public

## Overview

Learn the practice of documenting your learning journey, sharing works-in-progress, and open-sourcing projects publicly to build credibility and visibility in the AI community.

## Topics Covered

- Documenting your learning journey transparently
- Sharing works-in-progress and lessons learned
- Open-sourcing capstone projects on GitHub
- Creating thought leadership content

## Key Takeaways

- Building in public builds credibility that a resume alone cannot provide
- Sharing failures and learnings is as valuable as sharing successes
- Consistent public contribution compounds over time into career opportunities
