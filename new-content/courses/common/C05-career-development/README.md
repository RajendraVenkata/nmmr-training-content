# C05: Career Development for AI Professionals

## Overview

A practical, career-focused course covering the essential non-technical skills that separate AI professionals who get hired and promoted from those who struggle to break in. This course spans portfolio building, interview preparation, personal branding, and strategies for thriving in your first 90 days on the job.

## Prerequisites

- Completion of at least one technical course (C01-C04) or equivalent AI project experience
- At least one completed AI project suitable for portfolio inclusion
- A GitHub account and basic Git proficiency

## Modules

| Module | Name | Lessons | Key Topics |
|--------|------|---------|------------|
| M01 | Portfolio Building | 3 | Production-grade projects, five key portfolio pieces, open-sourcing and documentation |
| M02 | Interview Preparation | 4 | Technical foundations review, EDA skills, storytelling framework, trade-off analysis |
| M03 | Personal Branding & Networking | 3 | Building in public, LinkedIn strategy, community engagement |
| M04 | Your First 90 Days | 4 | Mindset shift, overcommunication, documentation habits, relationship building |

## Key Tools & Technologies

- **Portfolio**: GitHub, README templates, architecture diagramming tools
- **Interview Prep**: Python, SQL, statistics fundamentals, system design patterns
- **Branding**: LinkedIn, Twitter/X, personal blog platforms
- **Visualization**: UMAP, PCA for embedding visualization
- **Community**: Discord, Slack, meetup platforms, open-source contribution workflows

## Learning Outcomes

- Build a portfolio of five production-grade AI projects that demonstrate system-level thinking and measurable outcomes
- Write clear, recruiter-friendly documentation and architecture decision records for open-source projects
- Review and reinforce technical foundations in Python, SQL, and statistics for AI interviews
- Apply the storytelling framework to explain technical decisions to both engineers and business stakeholders
- Analyze latency vs. cost vs. quality trade-offs and justify architecture decisions in system design interviews
- Develop a personal brand through building in public, LinkedIn content strategy, and community engagement
- Navigate the first 90 days on the job using overcommunication, documentation habits, and strategic relationship building
- Combat imposter syndrome by tracking wins and maintaining a running accomplishment log
