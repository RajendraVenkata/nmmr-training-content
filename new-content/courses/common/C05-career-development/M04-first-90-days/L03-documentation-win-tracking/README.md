# Documentation & Win Tracking

## Overview

Build habits of documenting every process and lesson learned, and maintaining a running log of accomplishments that serves as evidence for performance reviews and combats imposter syndrome.

## Topics Covered

- Documenting every task, process, and lesson learned
- Keeping a running log of accomplishments and contributions
- Preparing evidence for performance reviews
- Using documentation to combat imposter syndrome

## Key Takeaways

- Documentation makes you organized and helps future new hires on the team
- A win log is essential for performance reviews and salary negotiations
- Writing down accomplishments counters imposter syndrome with concrete evidence
