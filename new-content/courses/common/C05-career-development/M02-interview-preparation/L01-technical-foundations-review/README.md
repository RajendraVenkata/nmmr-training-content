# L01: Technical Foundations Review

## Overview

This lesson provides a focused review of the core technical skills tested in AI job interviews. You will revisit Python data libraries, advanced SQL patterns, and essential statistics concepts to ensure you can confidently handle technical screening rounds.

## Topics Covered

- Python mastery: pandas for data manipulation
- Python mastery: numpy for numerical computing
- Python mastery: polars for high-performance data processing
- SQL proficiency: window functions and ranking
- SQL proficiency: Common Table Expressions (CTEs) and recursive queries
- Statistics: hypothesis testing and p-values
- Statistics: confidence intervals and margin of error
- Statistics: bias vs. variance trade-off
- The modern AI stack baseline interviewers expect

## Key Takeaways

- Interviewers expect fluency in pandas, numpy, and increasingly polars -- not just awareness of their existence
- Window functions and CTEs are among the most commonly tested SQL patterns in data and AI interviews
- Statistics questions focus on practical interpretation (what does this p-value mean for our decision) rather than formula memorization
- Understanding the bias-variance trade-off is fundamental to explaining model behavior and debugging poor performance
- Knowing the modern AI stack baseline (LLM APIs, vector databases, orchestration frameworks) signals that you are current and production-aware
