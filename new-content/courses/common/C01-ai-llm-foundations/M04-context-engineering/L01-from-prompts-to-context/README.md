# From Prompt Engineering to Context Engineering

## Overview

Understand the paradigm shift from simple prompt engineering to context engineering - the practice of managing everything the model sees, including system instructions, retrieved documents, tool outputs, and memory states.

## Topics Covered

- The evolution beyond prompt engineering
- Managing everything the model sees as a cohesive system
- Context as architecture, not just wording
- The shift from crafting prompts to designing context flows

## Key Takeaways

- In 2026, prompt engineering has evolved into context engineering
- Context engineering involves managing system instructions, retrieved documents, tool outputs, and memory states
- Mastering context flows through multi-step workflows is a major career differentiator
