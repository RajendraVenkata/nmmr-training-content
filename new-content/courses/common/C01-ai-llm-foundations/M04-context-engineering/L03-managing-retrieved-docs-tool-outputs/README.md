# Managing Retrieved Documents & Tool Outputs

## Overview

Learn how to effectively inject retrieved context and tool outputs into prompts while managing context window budgets and prioritizing the most relevant information sources.

## Topics Covered

- Injecting retrieved documents into model context
- Formatting tool outputs for optimal model consumption
- Context window budget management and allocation
- Prioritizing and ranking context sources for relevance

## Key Takeaways

- How you format and inject retrieved context directly impacts model output quality
- Context window budgets must be managed carefully to avoid truncation of critical information
- Prioritizing context sources ensures the model focuses on the most relevant information
