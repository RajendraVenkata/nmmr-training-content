# M01: Programming Foundations for AI

## Overview

This module establishes the programming skills needed to build AI applications across three core languages. You will learn Python for AI backend development, SQL for data access and vector store querying, and TypeScript for building AI-powered user interfaces.

## Prerequisites

- Basic programming experience in at least one language
- Familiarity with command-line tools
- A code editor (VS Code recommended)

## Lessons

| Lesson | Name | Has Lab? | Key Topics |
|--------|------|----------|------------|
| L01 | Python for AI Development | Yes | Python environment setup, virtual environments, key libraries (pandas, numpy, polars), async patterns |
| L02 | SQL for Data Access | Yes | Window functions, CTEs, complex joins, querying vector stores |
| L03 | TypeScript for AI Applications | Yes | TypeScript basics for AI UIs, Node.js backend for AI, API integration patterns |

## Key Tools

- Python 3.11+, pip, venv/conda
- pandas, numpy, polars
- PostgreSQL / SQLite
- TypeScript, Node.js, npm
- VS Code with relevant extensions

## Learning Outcomes

- Set up and manage Python virtual environments for AI projects
- Use pandas, numpy, and polars for data manipulation in AI pipelines
- Write advanced SQL queries including window functions, CTEs, and vector store queries
- Build TypeScript-based UI components that integrate with AI APIs
- Apply async programming patterns for efficient AI service communication
- Structure a full-stack AI development environment across Python, SQL, and TypeScript
