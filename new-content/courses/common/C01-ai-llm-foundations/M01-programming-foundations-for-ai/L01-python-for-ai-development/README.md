# L01: Python for AI Development

## Overview

This lesson covers setting up a professional Python development environment tailored for AI work. You will learn to manage virtual environments, work with essential data libraries, and apply async programming patterns critical for AI application development.

## Topics Covered

- Python environment setup and version management
- Virtual environments with venv and conda
- Key libraries: pandas for data manipulation
- Key libraries: numpy for numerical computing
- Key libraries: polars for high-performance data processing
- Async programming patterns for AI services
- Dependency management and reproducible environments

## Lab: Setting Up a Python AI Development Environment

Build a fully configured Python development environment from scratch, install and verify key AI libraries, and write a small async data processing script that demonstrates pandas, numpy, and polars in action.

### Lab Objectives

- Create and activate a Python virtual environment
- Install and configure pandas, numpy, and polars
- Write data loading and transformation scripts using each library
- Implement an async function that processes data concurrently
- Generate a requirements.txt for reproducible environment setup

## Key Takeaways

- A properly configured virtual environment prevents dependency conflicts across AI projects
- pandas, numpy, and polars each serve distinct roles in AI data pipelines
- Async patterns are essential when calling external AI services to avoid blocking execution
- Reproducible environments (via requirements.txt or pyproject.toml) are critical for team collaboration
