# C01: AI & LLM Foundations

## Overview

A comprehensive foundational course covering the essential knowledge and skills required to build AI-powered applications. This course spans programming fundamentals, LLM architecture, API integration, context engineering, evaluation methodologies, and AI security -- providing the bedrock for all subsequent beginner and intermediate tracks.

## Prerequisites

- Basic programming experience in at least one language
- Familiarity with command-line tools and Git
- Understanding of basic web concepts (HTTP, REST, JSON)

## Modules

| Module | Name | Lessons | Key Topics |
|--------|------|---------|------------|
| M01 | Programming Foundations for AI | 3 | Python, SQL, TypeScript for AI development |
| M02 | LLM Fundamentals | 4 | Transformers, tokenization, training lifecycle, model architectures |
| M03 | Working with LLM APIs | 3 | API calls, function calling, provider abstraction |
| M04 | Context Engineering | 4 | System instructions, retrieved context management, multi-stage flows |
| M05 | Evaluation & Reliability | 4 | Eval metrics, golden datasets, Ragas/TrueLens, CI/CD gating |
| M06 | AI Security & Safety | 3 | Prompt injection, PII handling, data leakage controls |

## Key Tools & Technologies

- **Languages**: Python, SQL, TypeScript
- **AI SDKs**: OpenAI SDK, Anthropic SDK, Groq SDK
- **Libraries**: pandas, numpy, polars, Pydantic, LiteLLM
- **Evaluation**: Ragas, TrueLens
- **Infrastructure**: Virtual environments, Node.js, CI/CD pipelines
- **Security**: Input sanitization libraries, PII detection tools

## Learning Outcomes

- Set up professional AI development environments in Python, SQL, and TypeScript
- Explain how transformer architectures and self-attention mechanisms power modern LLMs
- Make API calls to multiple LLM providers and parse structured responses
- Design effective system instructions and manage context windows for optimal model performance
- Build golden evaluation datasets and run automated quality assessments using Ragas and TrueLens
- Implement prompt injection defenses, PII handling, and data leakage controls in AI applications
- Apply provider abstraction patterns to build vendor-agnostic AI systems
- Wire evaluation gates into CI/CD pipelines to prevent quality regressions
