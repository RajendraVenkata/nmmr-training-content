# Evaluation Frameworks - Ragas & TrueLens

## Overview

Use production evaluation frameworks like Ragas and TrueLens to programmatically measure faithfulness, groundedness, and other quality metrics for RAG and AI systems.

## Topics Covered

- Ragas framework setup and configuration
- TrueLens integration for evaluation pipelines
- Measuring faithfulness and groundedness programmatically
- Building automated evaluation pipelines

## Lab: Running Automated Evaluations with Ragas

Set up and run a full Ragas evaluation suite on a RAG pipeline, measuring faithfulness and groundedness against a golden dataset.

### Lab Objectives

- Set up Ragas for evaluation of a RAG pipeline
- Run faithfulness and groundedness evaluations
- Generate evaluation reports with actionable metrics
- Compare results across different pipeline configurations

## Key Takeaways

- Ragas provides out-of-the-box metrics for RAG system evaluation
- Automated evaluation replaces manual testing and enables continuous quality monitoring
- Programmatic metrics enable data-driven decisions about system changes
