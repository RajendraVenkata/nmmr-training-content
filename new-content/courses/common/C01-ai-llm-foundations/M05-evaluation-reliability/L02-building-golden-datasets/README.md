# Building Golden Evaluation Datasets

## Overview

Learn to curate golden evaluation datasets of 50-200 manually verified question-answer pairs that serve as the ground truth for measuring AI system quality over time.

## Topics Covered

- Designing golden datasets for AI evaluation
- Curating 50-200 verified question-answer pairs
- Annotation guidelines and quality control
- Versioning and maintaining evaluation datasets

## Lab: Creating a Golden Evaluation Dataset

Design and create a golden dataset of verified Q&A pairs for evaluating a domain-specific AI system.

### Lab Objectives

- Design a golden dataset schema for a specific domain
- Create 50+ manually verified question-answer pairs
- Establish annotation guidelines for consistency
- Set up dataset versioning for tracking changes over time

## Key Takeaways

- Golden datasets are the foundation of systematic AI evaluation
- 50-200 verified pairs provide statistically meaningful quality measurement
- Annotation guidelines ensure consistency across evaluators and over time
